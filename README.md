# BITStream - Mood-Based Streaming Platform

BITStream is a full-stack web application built for the **BITS-RMIT program**. It allows users to stream content based on their current "vibe" or mood, featuring a robust PostgreSQL backend and a React-based frontend.

## 🚀 Features
- **Google OAuth 2.0:** Secure user authentication.
- **Mood-Based Filtering:** Dynamically filter content by Chill, Hype, Focus, etc.
- **Full CRUD Support:** Users can Upload, View, Search, and Delete videos.
- **View Analytics:** Real-time tracking of video popularity.
- **Responsive UI:** Dark-themed, mobile-friendly design using Tailwind CSS.

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS, React Router, Axios.
- **Backend:** Node.js, Express.js.
- **Database:** PostgreSQL.
- **Authentication:** Passport.js (Google Strategy).

## 🏃 Setup Instructions
1. **Database:** Run the SQL schema in your PostgreSQL terminal.
2. **Backend:** Run `npm install` and `node server.js` in the `/backend` folder.
3. **Frontend:** Run `npm install` and `npm run dev` in the `/frontend` folder.
